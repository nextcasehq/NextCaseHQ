# Pending Integration Register

This register tracks deferred integration, testing, and optimization work for NextCaseHQ Modules 13–20.

## ⚠ Critical Production Blocker

**Admin panel authentication is not real.** `apps/web/src/app/admin/page.tsx` checks the "password" and writes the `NEXTCASE_ADMIN_TOKEN` cookie entirely client-side — both the accepted password (one valid value is literally the word `admin`) and the token value are hardcoded in code shipped to every browser. PR #63 hardened the *server-side* comparison (env-configurable, constant-time, startup-validated) but did not and could not fix the client UI itself — that requires a real server-verified login endpoint, which is the next approved milestone (Server-Verified Admin Authentication). **Do not treat the admin console as production-safe until that milestone lands.**
